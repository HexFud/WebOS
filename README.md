# WebOS Demo

Piccola demo di una "WebOS": desktop con icone, dock e finestre semplici.

Come provare in locale:

1. Apri un terminale nella cartella del progetto (`/home/hexfud/Hackclub/webos`).
2. Avvia un server statico rapido:

```bash
python3 -m http.server 8000
```

3. Apri il browser su `http://localhost:8000` e clicca/doppio clic sulle icone.

File principali creati:
- index.html
- style.css
- app.js
- assets/icons/*.svg

Se vuoi, posso aggiungere altre app (mail, impostazioni, file manager), icone migliori o il drag-and-drop delle icone.
